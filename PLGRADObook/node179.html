JavaScript en el Lado del Servidor

Node es un intérprete JavScript escrito en C++ y con una API ligada a Unix para trabajar con procesos, fichero, sockets, etc. Los programas Node por defecto nunca se bloquean. Node utiliza manejadores de eventos que a menudo se implementan haciendo uso de funciones anidadas y clausuras.
