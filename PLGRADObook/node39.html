Práctica: Inventando un Lenguaje: Tortoise

El objetivo de esta práctica es crear un lenguaje de programación imperativa sencillo de estilo LOGO. Para ello lea el capítulo Inventing a Language - Tortoise del curso PL101: Create Your Own Programming Language de Nathan Whitehead. Haga todos los ejercicios e implemente el lenguaje descrito.

Puede encontrar una solución a la práctica en GitHub en el repositorio pl101 de Dave Ingram. Usela como guía cuando se sienta desorientado.
